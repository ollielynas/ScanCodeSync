
A singular progressive web app could be used to provide the 3 main functions required during a shoot. 

Using toggles the PWA could act as the [Director Metadata Recorder](Director%20Metadata%20Recorder.md), the [Operator Metadata Generator](Operator%20Metadata%20Generator.md), and as a master clock, or any combination of those functions.
